# Node命令及相关工具

## 命令



```shell
npm init -y  # 生成一份最简单的 package.json 文件

npm install express utility --save  # 安装依赖 express、utility 并自动把这些依赖写入 package.json 的 dependencies
```





## 工具

```shell
express
utility
eventproxy
superagent
cheerio
fs  #文件管理
path #处理路径
http #已废弃，推荐使用axios
nrm 
nodemon #代码热部署（全局安装），使用nodemon x.js 来代替node x.js 启动项目
```
