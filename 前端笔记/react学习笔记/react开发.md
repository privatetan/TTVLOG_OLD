# React开发

## 图片

##### 引入图片

```js
import backgroud from './images/backgroud1.JPG';
```

##### 引入图片

```jsx
<img src={backgroud} />
```

##### 添加为背景图片

```jsx
<div style={{ backgroundImage:`url(${backgroud})` }}>
```
